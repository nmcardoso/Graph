# Graph
This repository contains the graph data structure (directed graph, more specifically) and some algorithms.

Implementation languages:
* Python 3
* Ruby 2.2.0

## Running
Python 3:
```
$ python3 digraph.py
```
or
```
$ ./digraph.py
```

Ruby 2:
```
$ ruby digrapy.rb
```
or
```
$ ./digraph.rb
```
